O que é um Algoritmo?
- Sequência de instruções (passo-a-passo) para realizar uma determinada tarefa.


Comandos VISUALG

Escreva -> Mostra algo na tela   escreva()
Escreval -> Mostra algo na tela e faz quebra de linha escreval()
	Ex:  escreva("Olá mundo!")      escreva(2+2)
leia -> Recebe uma informação do usuário. Devemos passar entre () a variável que irá receber o dado.
	Ex: leia(nome)


Tipos de variáveis
inteiro  -> Para trabalhar com números inteiros
caractere -> Para trabalhar com caracteres/textos


Para concatenar TEXTOS utilize o sinal +
Para concatenar INTEIROS utilize a vírgula ,
